## 一、Trae 是什么？

**Trae**（读音 /treɪ/）是**字节跳动于 2025 年 1 月正式发布**的 AI 原生集成开发环境（AI IDE）。它基于 **VS Code 内核**深度定制，在保留经典编辑器操作习惯的同时，将大模型能力融入到编码的每一个环节。

与传统的"编辑器 + AI 插件"模式不同，Trae 从底层架构就围绕 AI 协作设计，提供了三种人机协同模式：

| 模式 | 定位 | 一句话说明 |
|------|------|-----------|
| **Chat 模式** | 智能问答助手 | 选中代码即可提问，AI 解释、修复、优化，随叫随到 |
| **Builder 模式** | 项目脚手架 | 用自然语言描述需求，AI 自动生成完整项目结构与代码 |
| **Solo 模式** | 全自动工程师 | AI 自主规划任务、写代码、运行测试、部署上线 |

> 🔥 **重点高亮**：Trae 国内版（[www.trae.cn](https://www.trae.cn)）**内置豆包、DeepSeek、Kimi、GLM、Qwen 等多个国产大模型**，**无需 API Key，无需科学上网，个人用户完全免费！**

---

## 二、Trae 能做什么？

Trae 不仅是一个代码编辑器，更是一个 **AI 编程搭档**。以下是它的核心能力：

- **智能代码补全**：实时理解上下文，提供精准的代码建议和自动补全
- **自然语言生成代码**：用中文描述需求，AI 自动生成对应代码
- **Bug 智能修复**：选中报错代码，AI 分析原因并给出修复方案
- **代码解释与重构**：一键理解复杂代码逻辑，自动优化代码结构
- **项目脚手架搭建**：Builder 模式下，一句话生成完整的前端/后端项目
- **终端命令辅助**：在终端中直接用自然语言描述操作，AI 转换为命令
- **多模态输入**：支持上传设计图/截图，AI 直接生成对应 UI 代码
- **VS Code 插件兼容**：完整兼容 VS Code 插件生态，可从 VS Code/Cursor 一键迁移配置

## 四、安装教程

### 🪟 Windows 平台安装

#### 系统要求

| 项目 | 最低要求 |
|------|----------|
| 操作系统 | **Windows 10 / 11（64 位）** |
| 内存 | 建议 ≥ 8GB（推荐 16GB） |
| 存储空间 | 预留 **1GB 以上**可用空间 |
| 运行环境 | 需要 **Visual C++ Redistributable 2019+** |

#### 安装步骤

**第一步：运行安装程序**

下载完成后，找到 `Trae-CN-Setup-x64.exe` 安装包（约 100-220MB），**右键 → 以管理员身份运行**。

> 📸 **截图参考**：安装包文件图标，可在文件资源管理器中查看已下载的 `.exe` 文件。

**第二步：同意许可协议**

在弹出的安装向导中，阅读许可协议后，勾选 **「我同意许可条款和条件」**，点击 **「下一步」**。

**第三步：选择安装路径**

默认安装路径为：

```
C:\Users\<用户名>\AppData\Local\Programs\Trae
```

> 🔥 **建议**：保持默认路径即可。如需自定义路径，请**避免使用中文目录名和空格**，防止后续插件兼容性问题。

点击 **「下一步」** 继续。

**第四步：创建快捷方式**

勾选以下选项（建议全选）：

- ✅ **创建桌面快捷方式** —— 方便快速启动
- ✅ **添加到 PATH 环境变量** —— 可在终端中使用 `trae` 命令启动
- ✅ **将 Trae 注册为默认编辑器** —— 可选

> 🔥 **重点高亮**：务必勾选 **「添加到 PATH 环境变量」**！这样你就可以在终端中直接通过命令行启动 Trae。

**第五步：开始安装**

点击 **「安装」** 按钮，等待安装进度条完成（约 1-2 分钟）。

**第六步：完成安装**

安装完成后，勾选 **「启动 Trae」**，点击 **「完成」**。

### 🍎 macOS 平台安装

#### 系统要求

| 项目 | 最低要求 |
|------|----------|
| 操作系统 | **macOS 12.0（Monterey）及以上** |
| 芯片支持 | **Intel 和 Apple Silicon（M1/M2/M3/M4）** 全兼容 |
| 内存 | 建议 ≥ 8GB（推荐 16GB） |
| 存储空间 | 预留 **1GB 以上**可用空间 |

> ⚠️ **注意**：如果 macOS 版本低于 12.0，官网提供旧版 IDE（v3.5.25 以下）的独立下载链接。

#### 安装步骤

**方法一：DMG 镜像安装（推荐新手）**

**第一步：下载 DMG 文件**

访问 [https://www.trae.cn](https://www.trae.cn)，选择 macOS 版本下载，文件名为 `Trae-CN-darwin-arm64.dmg`（Apple Silicon）或 `Trae-CN-darwin-x64.dmg`（Intel）。

> 📸 **截图参考**：Safari 浏览器下载栏中显示的 `.dmg` 文件。

**第二步：打开 DMG 镜像**

双击下载的 `.dmg` 文件，系统会自动挂载镜像，弹出一个安装窗口。

**第三步：拖拽安装**

在弹出的窗口中，将左侧的 **Trae 图标** 拖入右侧的 **「应用程序（Applications）」文件夹**。

```
 ┌──────────────┐         ┌──────────────────────┐
 │              │         │                      │
 │   Trae 图标   │  ───▶  │    Applications      │
 │              │  拖入   │     文件夹            │
 │              │         │                      │
 └──────────────┘         └──────────────────────┘
```

> 📸 **截图参考**：经典的 macOS 拖拽安装界面，与大多数 Mac 应用安装流程一致。

**第四步：等待复制完成**

文件复制进度条走完后，Trae 即安装完成。此时可以推出 DMG 镜像（右键桌面上的 Trae 图标 → 推出）。

**第五步：首次启动与安全设置**

进入 **「启动台（Launchpad）」** 或 **「应用程序」** 文件夹，双击 Trae 图标。

如果系统弹出 **「无法验证开发者」** 的安全提示：

1. 打开 **「系统设置」** → **「隐私与安全性」**
2. 在页面底部找到 **「已阻止 Trae 打开」** 的提示
3. 点击 **「仍要打开」** 按钮
4. 输入系统密码确认

> 🔥 **重点高亮**：macOS 首次打开非 App Store 应用都会触发安全检查，这是正常现象，按上述步骤放行即可。

---

**方法二：Homebrew 命令行安装（推荐开发者）**

如果你已安装 [Homebrew](https://brew.sh)，可以使用终端一键安装：

```bash
brew install --cask trae-editor
```

等待命令执行完成，Trae 会自动安装到「应用程序」文件夹中。

## 五、首次启动与初始配置

### 第一步：选择界面主题与语言

首次启动 Trae 后，会进入欢迎设置页面：

- **颜色主题**：提供浅色（Light）/ 深色（Dark）/ 跟随系统等多个主题选项
- **界面语言**：国内版默认中文，也支持切换为英文

> 🔥 **提示**：推荐选择**深色主题**，长时间编码更护眼。后续可在 `设置 → 外观 → 颜色主题` 中随时更换。

> 📸 **截图参考**：Trae 首次启动的欢迎页面，包含主题和语言选择界面。

### 第二步：登录账号（重要！）

AI 功能需要登录后才能使用。国内版支持：

- **手机号 + 验证码登录**（最常用）
- **稀土掘金账号登录**

点击右上角的 **「登录」** 按钮，输入手机号，获取验证码后完成登录。

> 🔥 **重点高亮**：登录是完全免费的，不会收取任何费用。登录后所有 AI 模型开箱即用，**无需配置任何 API Key**。

> 📸 **截图参考**：登录弹窗界面，包含手机号输入框与验证码按钮。

### 第三步：导入配置（可选）

如果你之前使用 VS Code 或 Cursor，Trae 支持一键导入配置：

1. 打开设置（`Ctrl/Cmd + ,`）
2. 搜索「导入配置」
3. 选择从 VS Code / Cursor 导入
4. 勾选需要导入的扩展、设置、快捷键

或者使用命令面板：

```
Ctrl/Cmd + Shift + P → 输入 "Import" → 选择 "Trae: Import Profile"
```

### 第四步：切换到 Builder 模式（推荐首次体验）

安装完成后，默认进入常规编辑器界面。点击右侧面板的 **「Builder」** 标签，进入 Builder 模式。

> 📸 **截图参考**：Trae Builder 模式界面，左侧为对话区，右侧为代码编辑区。

### 第五步：与 AI 对话，开始你的第一次 AI 编程！

在 Builder 对话框中输入你的需求，例如：

```
帮我创建一个 Python Flask 的待办事项应用，
包含添加、删除、标记完成功能，
前端使用简单的 HTML + CSS
```

然后按 `Ctrl/Cmd + Enter` 发送，AI 会自动：

1. 分析需求并规划项目结构
2. 逐文件生成代码
3. 提供运行和调试的指导

> 📸 **截图参考**：Builder 模式下 AI 正在生成代码的界面。

### 快捷键一览

| 操作 | Windows | macOS |
|------|---------|-------|
| 命令面板 | `Ctrl + Shift + P` | `Cmd + Shift + P` |
| 快速打开文件 | `Ctrl + P` | `Cmd + P` |
| 打开设置 | `Ctrl + ,` | `Cmd + ,` |
| 打开终端 | `Ctrl + `` ` | `Cmd + `` ` |
| 内联 Chat（选中代码后） | `Ctrl + I` | `Cmd + I` |
| 侧边栏 Chat | `Ctrl + U` | `Cmd + U` |
| 向 AI 发送消息 | `Ctrl + Enter` | `Cmd + Enter` |

### 模型切换

在 Chat/Builde r 面板的顶部，可以自由切换当前使用的 AI 模型：

- **豆包 Seed-Code**：字节自研，中文理解能力强
- **DeepSeek-V3**：综合能力强，代码生成质量高
- **DeepSeek-R1**：擅长逻辑推理和复杂算法
- **Qwen3-Coder**：阿里通义千问编程专用模型
- **GLM-4**：智谱 AI 模型

> 🔥 **提示**：不同场景可以切换不同模型。例如写前端页面用豆包，复杂算法问题用 DeepSeek-R1。

### 终端命令辅助

在 Trae 内置终端中，可以用自然语言描述你想要的命令：

```bash
# 在终端中按 Ctrl+U 打开终端 Chat，输入：
"查看当前目录下所有 Python 文件的大小，按大小降序排列"
```

AI 会自动生成对应命令并填入终端：

```bash
ls -lhS *.py
```

## 七、常见问题排查

### Q1：Windows 安装后启动报错缺少 DLL？

**解决方案**：安装 Visual C++ Redistributable 2019+

下载地址：[微软官方 VC++ 运行库](https://aka.ms/vs/17/release/vc_redist.x64.exe)

下载后双击安装，重启电脑即可。

### Q2：macOS 提示「Trae 已损坏，无法打开」？

**解决方案**：这是 macOS Gatekeeper 安全机制导致，并非文件真的损坏。

关闭提示后，打开终端执行：

```bash
sudo xattr -rd com.apple.quarantine /Applications/Trae.app
```

输入密码后重新打开 Trae 即可。

### Q3：登录后 AI 功能无法使用？

**排查步骤**：

1. 检查网络连接是否正常
2. 确认使用的是国内版（[trae.cn](https://trae.cn)）而非国际版
3. 尝试退出登录后重新登录
4. 检查 Trae 是否需要更新到最新版本

### Q4：如何更新 Trae？

Trae 会自动检测更新。你也可以手动检查：

1. 打开 Trae
2. 点击左下角齿轮图标 → **「设置」**
3. 搜索 `update` → 点击 **「检查更新」**
